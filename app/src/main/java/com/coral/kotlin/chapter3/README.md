语法糖：指在计算机语言中添加的某种语法，这种语法对语言的功能并没有影响，但是更方便程序员使用。
简而言之，语法糖让程序更加简洁，有更高的可读性。

如 Java 中的语法糖，编写程序可以是非常简单的语法，但是反编译后，都是通过底层 api 进行实现的。
比如 String... args 可变参数，assert 断言
